# Quantigration Database Management System

## Overview
Designed and implemented a relational database solution for Quantigration's Return Merchandise Authorization (RMA) process. The project focused on database design, data management, SQL query development, and business data analysis.

## Technologies Used
- SQL
- MySQL
- MySQL Workbench
- Relational Databases

## Features
- Created Customers, Orders, and RMA tables
- Implemented primary and foreign key relationships
- Imported data from CSV files
- Performed data validation and management
- Executed joins, inserts, updates, and delete operations
- Generated reports and business insights

## Skills Demonstrated
- Database Design
- SQL Query Development
- Relational Database Modeling
- Data Analysis
- Data Integrity Management
- Business Intelligence Reporting

## Key Accomplishments
- Designed normalized database tables with relationships
- Created and managed foreign key constraints
- Queried customer, order, and return data
- Analyzed return trends by state and product category
- Developed recommendations based on business data findings

## Outcome
Successfully developed a relational database solution capable of storing, managing, and analyzing customer order and return information to support business decision-making.
