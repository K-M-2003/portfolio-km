# Mobile Services Application

## Overview
The Mobile Services Application is a Java-based project developed to manage contacts, tasks, and appointments for a mobile application. The project emphasizes software quality through validation rules, service-layer management, and comprehensive unit testing.

## Technologies Used
- Java
- JUnit 5
- Object-Oriented Programming (OOP)
- Eclipse IDE

## Features

### Contact Service
- Create and manage contact records
- Store unique contact IDs
- Validate contact information
- Update contact details while maintaining data integrity

### Task Service
- Create and manage tasks
- Assign unique task identifiers
- Update task names and descriptions
- Prevent duplicate task entries

### Appointment Service
- Schedule appointments
- Validate future appointment dates
- Store appointment descriptions
- Remove appointments by ID

## Testing & Validation
The project includes comprehensive JUnit tests designed to verify system functionality and data validation requirements.

Testing areas include:
- Positive Testing
- Negative Testing
- Boundary Value Testing
- Exception Testing
- Duplicate ID Validation
- Data Integrity Verification
