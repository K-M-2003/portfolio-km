# SkyHayo!

## Overview
SkyHayo! is a weather application currently being developed. The goal of the project is to provide users with an intuitive and visually appealing way to view weather conditions and forecasts.

- Multi-day forecasts
- Location-based weather information
- User-friendly interface
- Responsive design
