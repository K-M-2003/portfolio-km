# Course Advising System

## Overview
The Course Advising System is a C++ application designed to help students and advisors manage course information. The program loads course data from a file, organizes courses using a Binary Search Tree (BST), and allows users to view course information and prerequisite requirements.

## Technologies Used
- C++
- Binary Search Trees (BST)
- File Input/Output
- Data Structures & Algorithms
- Object-Oriented Programming

## Features
- Load course data from a CSV file
- Store course information in a Binary Search Tree
- Display all courses in alphanumeric order
- Search for a specific course
- Display prerequisite information
- Validate user input and menu selections

## Skills Demonstrated
- Data Structures & Algorithms
- Binary Search Tree Implementation
- Recursive Programming
- File Processing
- Object-Oriented Programming
- Problem Solving
- Software Design

## Technical Highlights
- Implemented a custom Binary Search Tree for efficient course storage and retrieval
- Utilized in-order tree traversal to display courses in sorted order
- Developed recursive insertion and search operations
- Processed external course data files using file streams and string parsing
- Managed prerequisite relationships between courses

## Outcome
Successfully developed a course planning application capable of organizing and searching academic course information while demonstrating the practical application of data structures and algorithm design principles.
