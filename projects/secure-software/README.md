# Secure Software Enhancement Project

## Overview
This project focused on improving the security of an existing Java application by implementing secure software development practices.

## Technologies Used
- Java
- SHA-256 Hashing
- HTTPS
- Java Keytool
- OWASP Dependency-Check

## Key Enhancements
- Implemented SHA-256 password hashing
- Configured HTTPS communication
- Generated and managed security certificates
- Identified and addressed dependency vulnerabilities
- Applied secure coding principles

## Skills Demonstrated
- Software Security
- Secure Coding Practices
- Java Development
- Vulnerability Assessment
- Security Testing
